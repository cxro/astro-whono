## 1.设置个人上下文

将您的个人编码风格、首选库和通用说明添加到您的全局上下文文件中。

`/.claude/claude.md`

## 2.初始化一个项目

导航到项目目录,让AI为您生成一个特定于项目的上下文文件。
`/init`

## 3.记录业务流程

让 `Claude` 解释您项目中的一个业务流程,然后保存解释以备将来参考。

## 4.使用计划模式

开始一个新的对话,并在计划模式下与Claude交谈,以在实现之前勾勒出新功能的大纲。
`Shift + Tab`

## 5.管理上下文

练习在完成子任务后使用 `/compact` ,并在始新的相关任务时使用 `/clear` 。
